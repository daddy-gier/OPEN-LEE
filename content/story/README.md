# Story — Nightshade Hollow

Drop story documents here. Start with the template:

- **`STORY_STRUCTURE_TEMPLATE.md`** — fill this in first. It's the master
  document OPEN-LEE uses to understand the full narrative arc.

## What to add here

- Chapter outlines (one file per chapter)
- Major event descriptions
- Ending conditions
- Dialogue scripts (for scripted scenes)
- Quest/mission briefs

## Naming convention

```
ch01_arrival.md
ch02_survival.md
event_riot_trigger.md
ending_escape.md
```

The `event_` prefix is important — story event IDs in the game map to
files with that prefix so OPEN-LEE can look them up automatically.
