# Audio — Nightshade Hollow

Drop audio briefs, VO scripts, and music direction documents here.

## Categories

### Music
- `ost_brief.md` — Overall soundtrack direction (genre, mood, references)
- `ch01_music.md` — Per-chapter music notes
- `combat_music.md` — Alert/danger state music
- `menu_theme.md`

### Ambient Sound
- `cell_block_ambient.md` — Distant shouting, dripping, metal clanks
- `yard_ambient.md` — Wind, gravel, guard calls
- `cafeteria_ambient.md` — Crowd murmur, trays clattering

### Voice Over
- `vo_direction.md` — Acting notes for each character
- Per-character VO scripts can live in `content/characters/` alongside the profiles

## Reference format for VO scripts

```
CHARACTER: Guard Holloway
SCENE: First encounter with player (ev_arrival_01)
TONE: Authoritative, slightly bored, hints at awareness of player

LINE_001: "New fish. Cell block C. Move."
LINE_002: "Eyes forward. You'll learn the rules or I'll teach them to you."
```
