# Lore — Nightshade Hollow

Drop world-building documents here. Anything that enriches the world without
being a specific character or location:

- **Prison history** — when was it built, what happened here before
- **The system** — how the criminal justice system works in this world
- **Factions** — prisoner gangs, guard cliques, warden's agenda
- **Myths & rumors** — things prisoners whisper about (true or not)
- **The dark secret** — what Nightshade Hollow is really hiding
- **Outside world** — what the world is like beyond the walls

Any format is fine (.md, .txt, .docx description). OPEN-LEE will read these
to give NPCs and world events authentic context.
