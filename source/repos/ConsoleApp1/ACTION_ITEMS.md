# OPEN-LEE v3.1.0

## Multi-AI Consensus Engine for Windows

### What is OPEN-LEE?
Fire Claude, ChatGPT, Mistral, Grok, OpenClaw, and LudusAI **all at the same time**. 
Each model processes your prompt independently. Then OPEN-LEE synthesizes their responses 
into one unified, comprehensive answer.

### Key Features
✅ **Parallel AI Processing** — All 6 models fire simultaneously
✅ **Intelligent Synthesis** — Claude merges unique insights from each model
✅ **GPU Monitoring** — Real-time cluster stats (Buddy System)
✅ **Query Logging** — Track all conversations with timestamps
✅ **Secure** — API keys never touch your browser
✅ **Production Ready** — Error recovery, cancellation, timeouts

### System Requirements
- Windows 7 or later (64-bit)
- 4 GB RAM (8 GB recommended)
- 500 MB free disk space
- Internet connection (for cloud models)

### Installation
1. Download: OPEN-LEE-3.1.0 Setup.exe
2. Double-click
3. Follow installation wizard
4. Launch from Desktop icon or Start Menu

### Usage
1. Open OPEN-LEE
2. Type your question
3. Click ▶ EXECUTE
4. Watch all 6 models respond
5. See synthesized answer

### Optional: Set API Keys
Create `.env` file in installation folder with:
```
ANTHROPIC_API_KEY=sk-ant-...
OPENAI_API_KEY=sk-...
MISTRAL_API_KEY=...
GROK_API_KEY=...
```
